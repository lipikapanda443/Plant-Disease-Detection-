# Plant-Disease-Detection-
Developed an intelligent plant disease detection system using Machine Learning and Deep Learning
techniques to classify diseases from leaf images.
Performed image preprocessing, data augmentation,and feature extraction to improve model performance. 
Trained a Convolutional Neural Network (CNN) to accurately identify healthy and diseased plants. 
Evaluated the model using accuracy and loss metrics, enabling early disease diagnosis and supporting precision agriculture through timely intervention and
improved crop management.
